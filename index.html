MPSTME GNU/Linux User Group

About GLUG

The MPSTME GNU/Linux User Group (GLUG) is a libre organisation which provides support and education for both the GNU/Linux novice and the aficionado. Primarily, the group meets in person, but support is also provided on-line through Facebook and Twitter.

GLUG is primarily focused on hobbyist users and professionals who are engaged in self- directed study. The GLUG is not a committee, but a group of fervent enthusiasts willing to promote GNU/Linux and spread awareness about libre software among the students and faculty in the institute.

We function as a group and there is no hierarchical classification of members in the GLUG. The GLUG aim to build a culture of self-motivated learning and professional software development on industry standard technologies.

Events

Few events conducted by us include:

Python Workshop

Python, one of the newer languages is emerging as an extremely powerful and easy to learn programming language.The workshop aimed to introduce the language to the fresh batch of students in the college and covered it’s basics

Linux InstallFest

The event was organized to introduce the Linux Operating System. The participants were explained how Linux is powerful and stable. They were encouraged to switch to Linux as their primary OS. Different versions of Linux were also distributed among them. Some members also installed Linux on their workstations.

Grand Theft Ruby

The event consisted of teaching the fundamentals of the coding language “Ruby”. After an intensive interactive session of 2 hours on the first day of the event, the participants were provided with a set of challenging problems to be completed post event to help them further.

The second day of the event was on “Ruby on Rails”.

Advanced CPP

Topics that were covered include: Namespaces, Virtual functions, File handling, Multi-file programs, Templates, Pointers and strings, Exception handling, STL, Bit Manipulation, Casts, enumerations,macros, Generating Random numbers, Memory Management, C++ processor and Multithreading.

Software Freedom Day

People prefer open source software to proprietary software for a number of reasons, including: control, training, security and stability. The workshop aimed to discuss free and open source alternatives to the paid/closed source software which people most commonly use and also to disassemble a laptop and explain the internal workings.

Bash Workshop

A good command line interface is a marvelously expressive way of communicating with a computer in much the same way the written word is for human beings. The workshop was an internal event. It aimed at introducing members of GLUG to the terminal emulator and covered basics such as what is a shell, navigation, exploring the system, manipulating files and directories and other basic commands.
